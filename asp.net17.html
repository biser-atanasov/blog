  1. Creating the Post Method

Once again, in the Post Method, we need to take care of several things.

  1. Don't forget to use [HttpPost] attribute.

  2. The method signature cannot be the same as the post method, se we need a different name (e.g. DeleteConfirmed).

  3. We need to add an attribute that tells our app that this method will correspond to the "Delete" ction.
